# Changelog

All notable changes to the Noodle VS Code Extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Initial release of the Noodle IDE extension for VS Code
- Full language support for Noodle (.nc, .noodle files) with syntax highlighting
- AI-powered code generation, analysis, and optimization features
- Integrated compiler and runtime support for Noodle code
- Advanced debugging capabilities with breakpoints and variable inspection
- Performance profiling tools for Noodle applications
- Distributed execution support for scalable Noodle programs
- Custom themes (Noodle Dark and Noodle Light)
- Comprehensive code snippets for common Noodle patterns
- IntelliSense support with auto-completion, hover information, and signature help
- Integrated workspace management for Noodle projects
- Problem matcher for compiler errors and warnings
- Custom icons for Noodle files, folders, and projects
- Keyboard shortcuts for common operations
- Context menu integration for Noodle files
- Activity bar with dedicated Noodle explorer

### Features
- **AI Assistant**: Intelligent code generation using OpenAI, Anthropic, or local AI models
- **Language Support**: Full syntax highlighting, code folding, and bracket matching
- **Compilation**: One-click compilation with configurable compiler paths
- **Debugging**: Full-featured debugger with variable inspection and call stack
- **Profiling**: Performance analysis tools to identify bottlenecks
- **Themes**: Custom light and dark themes optimized for Noodle development
- **Snippets**: Time-saving code snippets for common Noodle constructs
- **IntelliSense**: Smart code completion, parameter hints, and quick info

### Configuration
- AI provider selection (OpenAI, Anthropic, Local)
- AI model configuration with customizable parameters
- Compiler and runtime path settings
- Debug mode selection (full, production, development)
- Theme selection (auto, dark, light)
- Feature toggles for AI, debugging, profiling, and distributed execution

### Known Issues
- Large files (>1MB) may experience slight performance degradation during syntax highlighting
- AI features require an active internet connection for cloud-based providers
- Distributed execution feature requires additional setup of Noodle runtime environment
- Debugging of optimized bytecode may show unexpected variable names

### Documentation
- Comprehensive README with getting started guide

- Troubleshooting guide for common issues
- Contribution guidelines for developers
