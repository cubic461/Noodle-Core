# Native GUI IDE

This directory contains the native GUI implementation of the NoodleCore IDE using Python/Tkinter.

## Structure

- `src/nativegui/` - Main IDE application source code
- `launchers/` - Startup scripts and launchers for different platforms
- `docs/` - Documentation for the native GUI IDE
- `tests/` - Test files for the native GUI IDE

## Features

- **Real File System Integration** - Live file browsing, editing, and management
- **Resizable Panel Layout** - Drag-to-resize functionality for all panels
- **Professional IDE Interface** - VSCode-like layout with tabs and menus
- **Complete AI Integration** - Real API connections with specialized agents
- **Multi-language Support** - Python, JavaScript, HTML, NoodleCore, and more
- **Working Terminal** - Actual command execution with history

## Launching

See the `launchers/` directory for platform-specific startup scripts.
